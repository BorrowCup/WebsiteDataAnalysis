Reuse Time Distribution

The following histogram shows the time taken (in days) for a cup to get reused. It makes use of 2 consecutive sale or return records to calculate this time.

Due to missing records for sales, total number of uses are almost half that of returns. Sale and Return data source options are provided to allow users to corroborate the patterns.

Please Note: Instances where the cup was reused after 30 days have been boxed in the last bin. Data post 22nd January, 2020 has been used to build the chart.
