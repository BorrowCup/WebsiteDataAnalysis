Suspected Cup Loss

The following two bar charts shows the percentages of cups that are, one, suspected to have been lost, two, may still be in the system. It has been assumed that if a cup isn't borrowed or returned (no sale or return record) for 21 days or more, it will be considered lost.

The first chart below shows the number of uses cups had before they were lost. The second chart shows the rest of the cups and their uses. It can be seen that this chart matches the pattern of the Use Frequency bar chart, which is expected.

Due to return records being 99.9% accurate, there is negligible effect of missing sale records on the below charts as any cups without salescans must have been scanned on return. However, it is a possibility that some returned cups may have been sitting in the facility or at cafes for more than 21 days. Since this is a preliminary analysis on lost cups, such a possibility has been set aside and may be considered in a much thorough analysis later.

Please Note: Data post 22nd January, 2020 has been used to build these charts.
